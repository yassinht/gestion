Pour utiliser cette application, voici quelques manipulations
a faire:
-Avoir un SGBD avec phpMyAdmin inclu, sur son PC (XAMPP recommander)
-Activé mysql dans le SGBD (Port 3306)
-creer une Base de données du nom: "gestiontachesetudiant"
-Importer le fichier:"gestiontachesetudiant.sql" dans la base de données depuis PhpMyAdmin
-Installer en fin l'application (eb executant le setup.exe)
